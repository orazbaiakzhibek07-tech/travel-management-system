## Саяхаттау жоспарлау жүйесі
Backend веб-қосымшаға арналған топтық жоба.
## Команда мүшелері
- Оразбай Ақжібек
- Тиллабек Баян
## Жобаның сипаттамасы
Бұл жоба — пайдаланушыларға саяхаттарды жоспарлауға мүмкіндік беретін веб-қосымша. 
Жүйеде пайдаланушы тіркеле алады, жүйеге кіре алады, саяхат бағыттарын қосу, көру, өңдеу және өшіру мүмкіндіктері қарастырылған. 
Барлық деректер реляциялық дерекқорда сақталады.
## Негізгі мүмкіндіктер
- Пайдаланушыны тіркеу және кіру (аутентификация)
- Саяхат жоспарларын қосу, өңдеу, жою
- Саяхаттарды іздеу және сүзгілеу
- Деректерді сервер жағында тексеру
## Қолданылатын технологиялар
- Python (Flask)
- PostgreSQL / SQLite
- GitHub
